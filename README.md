# 🚀 Playwright Automation Journey 2026

![Playwright Tests](https://github.com/devi123saha/playwright-ai-automation-portfolio/actions/workflows/playwright.yml/badge.svg)

This repository contains a professional end-to-end testing framework built with **Playwright** and **TypeScript**.

## 🛠️ Tech Stack & Patterns
- **Language:** TypeScript
- **Framework:** Playwright
- **Pattern:** Page Object Model (POM) for maintainable code
- **CI/CD:** GitHub Actions for automated regression testing
- **API Testing:** Integration with REST APIs (JSONPlaceholder)

## 🏗️ Project Structure
- `/tests`: Functional and API test suites
- `/pages`: Page Object classes containing locators and actions
- `.github/workflows`: CI/CD pipeline configuration

## 🚦 How to Run
1. Install dependencies: `npm install`
2. Run tests: `npx playwright test`
3. View report: `npx playwright show-report`
