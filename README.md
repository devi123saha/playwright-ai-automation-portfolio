# 🚀 Playwright & AI Automation Portfolio

## 👤 Project Overview
This repository documents my transition from Manual Testing to **AI-Augmented Test Automation**. In the 2026 landscape, I am focusing on building resilient, high-speed test suites using Playwright and AI productivity tools.

## 🛠 Tech Stack & Tools
- **Engine:** Playwright (TypeScript)
- **Node.js:** v24.12.0
- **AI Assistants:** Gemini (for test strategy), Playwright Codegen
- **Version Control:** Git & GitHub

## 🧪 Scenarios Automated
### 1. Authentication Flow
- **Site:** `the-internet.herokuapp.com`
- **Tests:** Successful login, failed login (invalid credentials), and secure logout.
- **Skill Demonstrated:** Handling dynamic web elements and cross-browser testing (Chromium).

### 2. CRUD Operations
- **Site:** `demo.playwright.dev/todomvc`
- **Tests:** Adding, completing, and verifying task persistence.
- **Skill Demonstrated:** State verification and list management.

## 🏁 How to Run
1. Clone the repo: `git clone <your-repo-link>`
2. Install dependencies: `npm install`
3. Run tests: `npx playwright test --project=chromium`
